# Riad di Siena

An 18th-century riad two minutes from Jemaa el-Fna. Marrakech medina.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Database**: Supabase
- **Styling**: Tailwind CSS
- **Fonts**: Playfair Display + Inter
- **Hosting**: Vercel
- **Images**: Cloudinary
- **Email**: Resend
- **Bookings**: PayPal + Make.com webhook
